PROGRAMME PROVISOIRE - Journées d’Études : Piloter des événements scientifiques à l’ère des technologies intellectives

A 148, Université Paris 8, Saint Denis, France | du jeudi 26 mars 2026 au vendredi 27 mars 2026

Warning

Ce programme est toujours en cours de discussion avec les intervenants et le comité scientifique

NoteProcessus de génération du programme

Ce programme regroupe les propositions générées à partir des informations des auteurs présentes dans la base de données Scanr et compilées dans le site expériemental du Laboratoire Paragraphe pour plus de détails cf. Module Omeka S Scanr.

Pour chaque auteur, un prompt est généré et soumis à Google Gemini (cf. détails du raisonnement) pour créer au format Markdown Quarto une proposition en lien avec le contexte du colloque.

Le code et l’explication détaillée du processus sont accessibles sur github GitHub.
